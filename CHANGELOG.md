# 更新日志

## 1.0.2 — 2026-07-21

### 安全与隐私

- 覆盖 Word people/presence、移动范围修订和更多审阅身份字段。
- PDF XMP 改为小型安全白名单，移除未知身份、联系信息和编辑历史字段。
- 检测 PDF trailer 加密字典及 ODF manifest 加密声明并安全跳过。
- 清理 ODF 模板路径，且只在审阅上下文中处理 dc:creator/dc:date。
- 清除 PDF trailer ID；含嵌入附件的 PDF 明确要求人工复核。
- “全部属性”模式移除图片 ICC profile。
- 所有格式统一通过临时文件排他发布，杜绝同名竞态覆盖或误删。

### 工程与发布

- 单元测试由 13 项扩展至 20 项。
- 补齐 AGPL-3.0-only、第三方许可说明、隐私边界和 GitHub Actions。
- 更新正式版权、版本资源和源码地址。
